# 网络

## HTTP 和 HTTPS 的区别

| 区别         | HTTP      | HTTPS                    |
| ------------ | --------- | ------------------------ |
| URL          | `http://` | `https://`               |
| 端口         | 80        | 443                      |
| 安全         | 不安全    | 安全                     |
| OSI 网络模型 | 应用层    | 安全传输机制工作在传输层 |
| 加密         | 无法加密  | 对传输的数据加密         |
| 证书         | 无需证书  | CA 机构颁发的 SSL 证书   |
